**Install:**
1. git clone https://github.com/petrokulybaba/symfony-blog.git
2. git checkout dev
3. composer install
4. php bin/console d:d:c
5. php bin/console d:s:u -f
6. php bin/console d:f:l -n
7. php bin/console s:r
